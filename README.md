# lobstertrap.github.io

Source for [lobstertrap.org](https://lobstertrap.org) — the public website for the LobsterTrap open-source community.

Built with plain HTML, CSS, and JavaScript. Served by GitHub Pages.

## Local Development

Open `index.html` in a browser, or use any static file server:

```bash
python3 -m http.server 8000
```
